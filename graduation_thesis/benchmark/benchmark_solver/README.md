# vrplib
Library for vehicle routing problems

## Description
vrplib : pythonのプログラムとソルバの実行ファイル
examples : jupyter notebookによるいくつかの実行例

## Requirement
Python3, OS(Windows,Linux,Mac)

## Usage
まず，exampleに例を参照してください．
pythonのプログラムにdocstringを書いているので，ヘルプで参照できます．
